\# 🧠 MemoRa (메모라)



> \*\*"Don't just say it, MemoRa it."\*\*

> 나만의 로컬 AI 회의 비서 \& 아카이브 시스템 (CPU / GPU / NPU 지원)



!\[Project Status](https://img.shields.io/badge/Status-Active-green)

!\[Python](https://img.shields.io/badge/Python-3.10+-blue)

!\[License](https://img.shields.io/badge/License-MIT-yellow)



\*\*MemoRa\*\*는 흩어져 있는 음성 기록을 한곳에 모아 텍스트로 변환하고, 핵심을 요약하여 영구 보존하는 오픈소스 프로젝트입니다.

N100 미니 PC부터 고성능 워크스테이션까지, 당신의 하드웨어에 맞춰 최적의 성능을 냅니다.



\## ✨ 주요 기능

1\.  \*\*Zero-Config Web UI:\*\* 복잡한 설정 파일 없이 웹에서 클릭만으로 하드웨어 가속(CUDA/CPU) 설정.

2\.  \*\*Smart Sync Player:\*\* 텍스트를 검색하고 클릭하면, 정확히 그 시점의 목소리를 들려줍니다.

3\.  \*\*Auto-Archiving:\*\* 텔레그램이나 구글 드라이브에 파일을 넣기만 하면, 자동으로 압축/분석/저장됩니다.

4\.  \*\*Human-in-the-Loop:\*\* AI의 초안을 수정하면 요약과 태그도 자동으로 업데이트됩니다.
